# 📊 Student Performance Analytics Dashboard

## 📌 Project Overview
This project is a data analytics dashboard built using Python and Streamlit. It analyzes student performance based on marks, attendance, study behavior, and assignments. The dashboard provides visual insights to understand student performance trends easily.

---

## 📁 Dataset Information
The dataset used contains the following features:

- Student_ID
- Study_Hours
- Attendance
- Assignments
- Marks

It is a synthetic dataset generated using Python (NumPy & Pandas) for analysis purposes.

---

## 🤖 Model Used
No machine learning model is used in this dashboard.

However, basic data analysis techniques are applied:
- Statistical analysis (mean, average)
- Feature correlation analysis
- Performance categorization logic

---

## ⚙️ Installation Steps

### 1. Clone or download project

git clone <your-repo-link>


### 2. Install dependencies

pip install pandas numpy matplotlib seaborn streamlit


### 3. Run the dashboard

streamlit run dashboard.py

## 📊 Dashboard Features

- Total Students count
- Average Marks calculation
- Average Attendance calculation
- Attendance Distribution chart

- Feature Correlation Heatmap

- Dataset preview table
